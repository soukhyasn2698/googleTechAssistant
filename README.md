## TechHelperAgent

**TechHelperAgent** is a voice-based AI assistant designed to help users troubleshoot everyday technology issues. It guides users step-by-step through problems with Wi-Fi, Bluetooth, apps, and devices, making technology more accessible for beginners, elderly users, or anyone who struggles with modern tech.

---

##  What it does
- Listens to your voice input  
- Understands the tech issue  
- Provides **clear, step-by-step guidance**  
- Supports problems related to Wi-Fi, Bluetooth, apps, and devices  
- Designed for beginners and elderly users  

---
## Technologies Used

- **Google ADK** – Agent Development Kit for building intelligent agents  
- **Gemini Models (Gemini Live api)** – gemini-live-2.5-flash-preview-native-audio-09-2025 
- **FastAPI** – Backend API framework  
- **Python 3.10**  
- **Google Cloud Services**:
  - **Vertex AI** (for Gemini model execution)
  - **Cloud Run** (for hosting the agent backend)
